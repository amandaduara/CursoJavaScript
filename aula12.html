Retornando o tipo da variável

Em JavaScript, o operador typeof é usado para obter o tipo de dado de uma expressão ou variável. Ele retorna uma string que representa o tipo do operando. Alguns dos valores retornados são:

  • "number" para números.
  • "number" para números.
  • "string" para strings.
  • "boolean" para booleanos.
  • "undefined" para valores indefinidos.
  • "object" para números (incluindo arrays e nulos).
  • "function" para funções.
  • "symbol" para símbolos (introduzidos no ECMAScript 6).

Além disso, tenha em mente que o typeof é uma ferramenta útil, mas nem sempre fornece a precisão desejada em situações mais complexas. Em alguns casos, é preferível usar outros métodos para verificar o tipo de dado, como o operador instanceof para objetos específico
