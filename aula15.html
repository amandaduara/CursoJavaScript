Aula 15 - Loop FOR


O loop for é uma estrutura de controle de fluxo que permite repetir um bloco de código um número conhecido de vezes. A sintaxe básica do loop for varia um pouco entre as linguagens de programação, mas a ideia geral é a mesma.

O loop for é configurado com três partes:

for (let i = 0; i < 5; i++)

  • A expressão de inicialização (let i = 0): Inicializa a variável de controle (i) com um valor inicial.
  • A condição de continuação (i < 5): Determina se o loop deve continuar executando com base na condição especificada.
  • A expressão de incremento (i++): Atualiza a variável de controle após cada iteração.
Observação:
  1. Cada repetição do loop é chamado de iteração.
  2. Loops Definidos - FOR (Quando sei a quantidade de iterações que é preciso fzr)
  3. Loops Indefinidos - While, Do While (Quando não sei a quantidade de iterações que é preciso fzr)

O loop for é útil quando você sabe exatamente quantas iterações são necessárias. Ele simplifica o código e melhora a legibilidade em comparação com a repetição manual de código. Além disso, é comumente utilizado para percorrer elementos em uma sequência, como arrays ou listas.
