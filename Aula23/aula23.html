Aula 23 - Parâmetros REST em Funções


Em JavaScript, ao criar funções que utilizam parâmetros, podemos definir esses parâmetros de várias maneiras. Os parâmetros podem ser passados para uma função de forma convencional ou usando recursos específicos, como os parâmetros REST.

  • Parâmetros Convencionais: explicado durante a aula 22.
  • Parâmetros REST: O parâmetro REST permite que uma função aceite um número indefinido de argumentos como um array. Sua sintaxe é: function(a, b, ...c) { // código }

Devemos lembrar de que os parâmetros REST devem ser os últimos na lista de parâmetros da função. Além disso, a utilização de parâmetros REST é opcional, e você pode escolher qual abordagem faz mais sentido para sua situação específica.
