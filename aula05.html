Aula 5 - Operadores Relacionais


OPERADORES RELACIONAIS:

Através dos operadores relacionais podemos realizar uma comparação entre variáveis e obter o resultado de true ou false. Sendo eles:

  • < : mostra se um valor é menor que outro.
  • <= : se um valor é menor ou igual que outro.
  • > : compara se um valor é maior que o outro.
  • >= : se um valor é maior ou igual que outro.
  • == : mostra se os valores são iguais.
  • != : exibe se os valores são diferentes.

OPERADOR DE NEGAÇÃO:

O operador de negação nega o resultado de uma comparação, portanto se uma operação resultar em true ele negará e exibirá false.

  • !() : nega uma comparação.
