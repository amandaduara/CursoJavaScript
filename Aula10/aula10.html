Aula 10 - Operador Ternário


OPERADOR TERNÁRIO (?)

O operador ternário em JavaScript é uma forma concisa de expressar uma instrução condicional. Ele é frequentemente usado para simplificar estruturas if-else em uma única linha. A sintaxe do operador ternário é a seguinte:

condição ? expressão_se_verdadeira : expressão_se_falsa;

O operador ternário é útil para simplificar blocos de código condicional curtos e tornar o código mais legível quando a lógica é simples. No entanto, é importante usá-lo com moderação para evitar tornar o código muito complexo ou difícil de entender.
