Aula 19 - Declarações BREAK e CONTINUE


As declarações break e continue são instruções de controle de fluxo utilizadas em muitas linguagens de programação para modificar o comportamento de loops (como for, while e do-while).

Break
  • A declaração break é usada para sair imediatamente de um loop, interrompendo sua execução, mesmo que a condição do loop ainda seja verdadeira.
  • É útil quando você deseja encerrar prematuramente a execução de um loop com base em uma condição específica.
Continue
  • A declaração continue é usada para pular o restante do código no corpo do loop e avançar para a próxima iteração, mesmo que a condição do loop ainda seja verdadeira.
  • É útil quando você deseja ignorar parte do código em uma iteração específica do loop.

Em resumo, break é usado para sair completamente do loop, enquanto continue é usado para pular o restante do código no corpo do loop e passar para a próxima iteração. Ambas são ferramentas poderosas para controlar o fluxo de execução em estruturas de repetição.
